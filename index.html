Latest Projects

Calculator

This is a simple yet functional calculator built using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The user-friendly interface ensures smooth interaction, and the design is fully responsive, making it accessible across different devices. The calculator provides instant results with each input, offering a seamless experience. Future enhancements will include features like a history of calculations and a dark mode for better usability.

Tic-Tac-Toe

This is a classic Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game features a simple yet interactive user interface, allowing two players to take turns marking ‘X’ and ‘O’ on a 3x3 grid. It includes logic to detect the winner or declare a draw when all spaces are filled. The game provides instant feedback and highlights the winning combination. The design is fully responsive, ensuring compatibility across different devices. Future improvements may include an AI-based single-player mode and a reset feature for continuous gameplay.

Rock,Paper & Scissors

This is a simple and interactive Rock, Paper, Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, which randomly selects its move. The logic determines the winner based on traditional rules—rock beats scissors, scissors beat paper, and paper beats rock. The user-friendly interface displays the choices, results, and score updates dynamically. The design is fully responsive, ensuring smooth gameplay on different devices.
